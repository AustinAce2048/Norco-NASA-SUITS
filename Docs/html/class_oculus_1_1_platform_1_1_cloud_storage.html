static Request<Models.CloudStorageUpdateResponse> Oculus.Platform.CloudStorage.Save ( string  bucket,
string  key,
byte[]  data,
long  counter,
string  extraData 
)
static

Note: Cloud Storage is only available for Rift apps.

Send a save data buffer to the platform. CloudStorage.Save() passes a pointer to your data in an async call. You need to maintain the save data until you receive the message indicating that the save was successful.

If the data is destroyed or modified prior to receiving that message the data will not be saved.

Parameters
bucketThe name of the storage bucket.
keyThe name for this saved data.
dataStart of the data block.
counterOptional. Counter used for user data or auto-deconfliction.
extraDataOptional. String data that isn't used by the platform.

Error codes

  • 100: The stored version has a later timestamp than the data provided. This cloud storage bucket's conflict resolution policy is configured to use the latest timestamp, which is configurable in the developer dashboard.
