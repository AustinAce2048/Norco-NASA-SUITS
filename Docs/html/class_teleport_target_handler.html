Detailed Description

The TeleportTargetHandler's main purpose is to determine when the current aim target is valid and to update the teleport destination as required by the design.

This allows specialized versions that can simply update the destination to any arbitrary location, or update only when a teleport node is being pointed at, or anything else that fits the design for limiting when & where a teleport is valid.
